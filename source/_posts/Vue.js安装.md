#### 官网

```
https://cn.vuejs.org/
```

#### NPM 方法

```
淘宝镜像
	$ npm install -g cnpm --registry=https://registry.npm.taobao.org
查看版本
	$ npm -v
升级 npm
	cnpm install npm -g
升级或者安装 cnpm	
	npm install cnpm -g
在用Vue.js构建大型应用时推荐使用npm安装:
	$ cnpm install vue
```

#### 命令行工具

```
全局安装 vue-cli
	$ cnpm install --global vue-cli
创建一个基于webpack 模板的新项目
	$ vue init webpack my-project
进入项目，安装并运行
	$ cd my-project 
	$ cnpm install 
	$ cnpm run dev
```
